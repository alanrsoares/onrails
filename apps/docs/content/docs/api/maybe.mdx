# @onrails/maybe API Reference

## Functions

### andThen

Alias for `flatMap` ŌĆö same dual-form shape.

```typescript
function andThen<T, U>(maybe: Maybe<T>, fn: (value: T) => Maybe<U>): Maybe<U>
function andThen<T, U>(fn: (value: T) => Maybe<U>): (maybe: Maybe<T>) => Maybe<U>
```

### compact

Drop every `None` and collect the remaining `Some` values into an array.

```typescript
function compact<T>(maybes: readonly Maybe<T>[]): T[]
```

#### Example

```ts
compact([some(1), none(), some(2)]);    // [1, 2]
```

### compactMap

`compact(items.map(fn))` ŌĆö map each item to `Maybe`, drop `None`,
collect the rest.

```typescript
function compactMap<T, U>(items: readonly T[], fn: (item: T) => Maybe<U>): U[]
```

#### Example

```ts
compactMap(rawRows, (row) =>
  row.deletedAt ? none() : some(row.id),
);
// string[]
```

### flatMap

Canonical bind for `Maybe`. Chains a `Maybe`-returning step;
short-circuits on `None`. Dual-form.

```typescript
function flatMap<T, U>(maybe: Maybe<T>, fn: (value: T) => Maybe<U>): Maybe<U>
function flatMap<T, U>(fn: (value: T) => Maybe<U>): (maybe: Maybe<T>) => Maybe<U>
```

#### Example

```ts
flatMap(fromNullable(user), (u) =>
  u.active ? some(u) : none(),
);
```

### fromNullable

Lift a nullable value into `Maybe<T>` ŌĆö `null` and `undefined` become
`None`; anything else becomes `Some`.

```typescript
function fromNullable<T>(value: T | null | undefined): Maybe<T>
```

#### Example

```ts
fromNullable(map.get(key));     // Maybe<V>
fromNullable(null);             // None
```

### getOrElse

Returns the `Some` value, or `defaultValue` when the value is `None`.

```typescript
function getOrElse<T>(maybe: Maybe<T>, defaultValue: T): T
```

### isNone

Type-narrowing predicate: returns `true` when the value is `None`.

```typescript
function isNone<T>(maybe: Maybe<T>): maybe is { readonly _tag: "None"; }
```

### isSome

Type-narrowing predicate: returns `true` when the value is `Some`.

```typescript
function isSome<T>(maybe: Maybe<T>): maybe is Some<T>
```

#### Example

```ts
if (isSome(m)) {
  console.log(m.value);   // narrowed to Some branch
}
```

### map

Transform the `Some` value, passing `None` through unchanged. Dual-form.

```typescript
function map<T, U>(maybe: Maybe<T>, fn: (value: T) => U): Maybe<U>
function map<T, U>(fn: (value: T) => U): (maybe: Maybe<T>) => Maybe<U>
```

#### Example

```ts
map(some(2), (n) => n * 3);                      // Some 6 ŌĆö data-first
pipe(fromNullable(input), map((s) => s.trim())); // curried
```

### match

Terminal collapse ŌĆö fold both branches into a single value. Positional,
dual-form. Same shape as `Result.match` for sibling consistency.

For files that also import `match` from `ts-pattern`, the alias
`matchMaybe` is identical.

```typescript
function match<T, U>(maybe: Maybe<T>, onSome: (value: T) => U, onNone: () => U): U
function match<T, U>(onSome: (value: T) => U, onNone: () => U): (maybe: Maybe<T>) => U
```

#### Example

```ts
const greeting = match(
  fromNullable(user),
  (u) => `hello ${u.name}`,
  () => "hello guest",
);
```

### matchMaybe

Collision-free alias ŌĆö mirrors `matchResult` on `@onrails/result`.

```typescript
function matchMaybe<T, U>(maybe: Maybe<T>, onSome: (value: T) => U, onNone: () => U): U
function matchMaybe<T, U>(onSome: (value: T) => U, onNone: () => U): (maybe: Maybe<T>) => U
```

### none

The `None` value ŌĆö represents expected absence. Not a failure.

```typescript
function none<T = never>(): Maybe<T>
```

#### Example

```ts
const cached: Maybe<User> = cache.has(id) ? some(cache.get(id)) : none();
```

### of

Fantasy Land `of` ŌĆö alias of `some`.

```typescript
function of<T>(value: T): Maybe<T>
```

### optional

`flatMap(fromNullable(value), fn)` ŌĆö lift a nullable then bind through
a `Maybe`-returning step. Convenient for chaining lookups.

```typescript
function optional<T, U>(value: T | null | undefined, fn: (value: T) => Maybe<U>): Maybe<U>
```

#### Example

```ts
optional(rawId, (id) => fromNullable(cache.get(id)));
```

### some

Lifts a value into the `Some` branch.

```typescript
function some<T>(value: T): Maybe<T>
```

#### Example

```ts
const m = some(1);   // Maybe<number>
```

### tap

Run a side effect on the `Some` value, pass the `Maybe` through unchanged.
`None` is a no-op. Mirrors `tap` from `@onrails/result`. Dual-form.

```typescript
function tap<T>(maybe: Maybe<T>, fn: (value: T) => void): Maybe<T>
function tap<T>(fn: (value: T) => void): (maybe: Maybe<T>) => Maybe<T>
```

#### Example

```ts
tap(some(2), (n) => console.log(n));               // logs 2, returns Some 2
pipe(fromNullable(row), tap((r) => sink.push(r))); // curried ŌĆö collect on Some
```

### tapNone

Run a side effect when the `Maybe` is `None`, pass it through unchanged.
`Some` is a no-op. The `None`-side mirror of `tap`. Dual-form.

```typescript
function tapNone<T>(maybe: Maybe<T>, fn: () => void): Maybe<T>
function tapNone<T>(fn: () => void): (maybe: Maybe<T>) => Maybe<T>
```

#### Example

```ts
tapNone(none(), () => metrics.miss());              // runs effect, returns None
pipe(fromNullable(hit), tapNone(() => metrics.miss())); // curried
```

### unwrap

Unwraps the `Some` value or throws if the value is `None`.

```typescript
function unwrap<T>(maybe: Maybe<T>): T
```

#### Example

```ts
expect(unwrap(parseUser(raw))).toEqual(expected);
```

### unwrapOr

Returns the `Some` value, or `defaultValue` when the value is `None`.

```typescript
function unwrapOr<T>(maybe: Maybe<T>, defaultValue: T): T
```

#### Example

```ts
unwrapOr(fromNullable(user), { name: "guest" });
```

## Types

### Maybe

Tagged optional value ŌĆö absence is not failure.

```typescript
export type Maybe<T> = { readonly _tag: "Some"; readonly value: T } | { readonly _tag: "None" };
```

### None

```typescript
export type None = Extract<Maybe<never>, { _tag: "None" }>;
```

### Some

```typescript
export type Some<T> = Extract<Maybe<T>, { _tag: "Some" }>;
```
