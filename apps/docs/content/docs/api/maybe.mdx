# @onrails/maybe API Reference

## Constructors

### none <span className="inline-flex items-center rounded-full border border-emerald-200 dark:border-emerald-800 bg-emerald-50 dark:bg-emerald-950/30 text-emerald-600 dark:text-emerald-400 text-xs font-semibold px-2.5 py-0.5 font-medium ml-2 align-middle">ƒ</span>

The `None` value — represents expected absence. Not a failure.

```typescript
function none<T = never>(): Maybe<T>
```

#### Example

```ts
const cached: Maybe<User> = cache.has(id) ? some(cache.get(id)) : none();
```

Lifts a value into the `Some` branch.

```typescript
function some<T>(value: T): Maybe<T>
```

#### Parameters

| Parameter | Type | Description |
|---|---|---|
| `value` | `T` |  |

#### Example

```ts
const m = some(1);   // Maybe<number>
```

## Core

### andThen <span className="inline-flex items-center rounded-full border border-emerald-200 dark:border-emerald-800 bg-emerald-50 dark:bg-emerald-950/30 text-emerald-600 dark:text-emerald-400 text-xs font-semibold px-2.5 py-0.5 font-medium ml-2 align-middle">ƒ</span>

Neverthrow-compat alias of [flatMap](./result#flatmap-ƒ) — same dual-form shape. Prefer
[flatMap](./result#flatmap-ƒ) as the canonical name; reach for `andThen` only when porting
neverthrow code or matching its vocabulary.

```typescript
function andThen<T, U>(maybe: Maybe<T>, fn: (value: T) => Maybe<U>): Maybe<U>
function andThen<T, U>(fn: (value: T) => Maybe<U>): (maybe: Maybe<T>) => Maybe<U>
```

#### Parameters

| Parameter | Type | Description |
|---|---|---|
| `maybe` | `Maybe<T>` |  |
| `fn` | `(value: T) => Maybe<U>` |  |

#### Example

```ts
andThen(fromNullable(user), (u) =>
  u.active ? some(u) : none(),
);
```

<hr className="my-8 border-neutral-200 dark:border-neutral-800" />

### flatMap <span className="inline-flex items-center rounded-full border border-emerald-200 dark:border-emerald-800 bg-emerald-50 dark:bg-emerald-950/30 text-emerald-600 dark:text-emerald-400 text-xs font-semibold px-2.5 py-0.5 font-medium ml-2 align-middle">ƒ</span>

```typescript
function flatMap<T, U>(fn: (value: T) => Maybe<U>): (maybe: Maybe<T>) => Maybe<U>
```

#### Parameters

| Parameter | Type | Description |
|---|---|---|
| `maybe` | `Maybe<T>` |  |
| `fn` | `(value: T) => Maybe<U>` |  |

#### Example

```ts
flatMap(fromNullable(user), (u) =>
```

### map <span className="inline-flex items-center rounded-full border border-emerald-200 dark:border-emerald-800 bg-emerald-50 dark:bg-emerald-950/30 text-emerald-600 dark:text-emerald-400 text-xs font-semibold px-2.5 py-0.5 font-medium ml-2 align-middle">ƒ</span>

Transform the `Some` value, passing `None` through unchanged. Dual-form.

```typescript
function map<T, U>(maybe: Maybe<T>, fn: (value: T) => U): Maybe<U>
function map<T, U>(fn: (value: T) => U): (maybe: Maybe<T>) => Maybe<U>
```

#### Parameters

| Parameter | Type | Description |
|---|---|---|
| `maybe` | `Maybe<T>` |  |
| `fn` | `(value: T) => U` |  |

#### Example

```ts
map(some(2), (n) => n * 3);                      // Some 6 — data-first
pipe(fromNullable(input), map((s) => s.trim())); // curried
```

<hr className="my-8 border-neutral-200 dark:border-neutral-800" />

### match <span className="inline-flex items-center rounded-full border border-emerald-200 dark:border-emerald-800 bg-emerald-50 dark:bg-emerald-950/30 text-emerald-600 dark:text-emerald-400 text-xs font-semibold px-2.5 py-0.5 font-medium ml-2 align-middle">ƒ</span>

Terminal collapse — fold both branches into a single value. Positional,
dual-form. Same shape as `Result.match` for sibling consistency.

For files that also import `match` from `ts-pattern`, use a namespace
import (`import * as Maybe from "@onrails/maybe"` → `Maybe.match`) to
dissolve the collision.

```typescript
function match<T, U>(maybe: Maybe<T>, onSome: (value: T) => U, onNone: () => U): U
function match<T, U>(onSome: (value: T) => U, onNone: () => U): (maybe: Maybe<T>) => U
```

#### Parameters

| Parameter | Type | Description |
|---|---|---|
| `maybe` | `Maybe<T>` |  |
| `onSome` | `(value: T) => U` |  |
| `onNone` | `() => U` |  |

#### Example

```ts
const greeting = match(
  fromNullable(user),
  (u) => `hello ${u.name}`,
  () => "hello guest",
);
```

### tap <span className="inline-flex items-center rounded-full border border-emerald-200 dark:border-emerald-800 bg-emerald-50 dark:bg-emerald-950/30 text-emerald-600 dark:text-emerald-400 text-xs font-semibold px-2.5 py-0.5 font-medium ml-2 align-middle">ƒ</span>

Run a side effect on the `Some` value, pass the `Maybe` through unchanged.
`None` is a no-op. Mirrors `tap` from `@onrails/result`. Dual-form.

```typescript
function tap<T>(maybe: Maybe<T>, fn: (value: T) => void): Maybe<T>
function tap<T>(fn: (value: T) => void): (maybe: Maybe<T>) => Maybe<T>
```

#### Parameters

| Parameter | Type | Description |
|---|---|---|
| `maybe` | `Maybe<T>` |  |
| `fn` | `(value: T) => void` |  |

#### Example

```ts
tap(some(2), (n) => console.log(n));               // logs 2, returns Some 2
pipe(fromNullable(row), tap((r) => sink.push(r))); // curried — collect on Some
```

<hr className="my-8 border-neutral-200 dark:border-neutral-800" />

### tapNone <span className="inline-flex items-center rounded-full border border-emerald-200 dark:border-emerald-800 bg-emerald-50 dark:bg-emerald-950/30 text-emerald-600 dark:text-emerald-400 text-xs font-semibold px-2.5 py-0.5 font-medium ml-2 align-middle">ƒ</span>

Run a side effect when the `Maybe` is `None`, pass it through unchanged.
`Some` is a no-op. The `None`-side mirror of [tap](#tap-ƒ). Dual-form.

```typescript
function tapNone<T>(maybe: Maybe<T>, fn: () => void): Maybe<T>
function tapNone<T>(fn: () => void): (maybe: Maybe<T>) => Maybe<T>
```

#### Parameters

| Parameter | Type | Description |
|---|---|---|
| `maybe` | `Maybe<T>` |  |
| `fn` | `() => void` |  |

#### Example

```ts
tapNone(none(), () => metrics.miss());              // runs effect, returns None
pipe(fromNullable(hit), tapNone(() => metrics.miss())); // curried
```

## Utilities

|---|---|---|

## Types

### Maybe <span className="inline-flex items-center rounded-full border border-teal-200 dark:border-teal-800 bg-teal-50 dark:bg-teal-950/30 text-teal-600 dark:text-teal-400 uppercase tracking-wider px-2 py-0.5 text-[10px] font-medium ml-2 align-middle">type</span>

Tagged optional value — a discriminated union over `_tag` where absence
(`None`) is an expected outcome, not a failure. Construct with [some](#some-ƒ) /
[none](#none-ƒ), narrow with [isSome](#issome-ƒ) / [isNone](#isnone-ƒ), and collapse with
[match](./result#match-ƒ).

```typescript
export type Maybe<T> = { readonly _tag: "Some"; readonly value: T } | { readonly _tag: "None" };
```

#### Example

```ts
function findUser(id: string): Maybe<User> {
  return fromNullable(db.users.get(id));
}
```
